# Roadmap

## Reliability

- Add longer-running scoring evaluation with anonymized synthetic traces.
- Add explicit recommendation-change explanations in the UI.
- Add configurable recommendation hysteresis and recent-window sizes.

## Compatibility

- Expand real-device validation across supported v2rayN versions.
- Add safe, documented capability probes for more Mihomo desktop clients.
- Keep sing-box integrations read-only until stable control APIs are available.

## Maintainer workflow

- Publish signed release checksums.
- Add sanitized compatibility fixtures contributed by users.
- Track client/version confirmations in public issues.
