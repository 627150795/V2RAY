## ProxyMonitor 是什么？

ProxyMonitor 是一个 Windows 节点健康监控工具，用来长期跟踪 v2rayN、Mihomo / Clash、sing-box 等客户端里的代理节点表现。

它会定期用低流量方式测试节点延迟、稳定性和轻量速度，保存本地历史，并推荐：

- 最稳定节点
- 速度最快节点
- 综合最佳节点

它不会上传订阅、节点、测速历史，也不会自动切换节点。切换节点需要用户确认。

## 适合谁？

适合订阅节点很多、经常不知道该选哪个节点、希望用长期数据而不是单次测速结果判断节点质量的 Windows 用户。

## 下载

普通用户下载 `ProxyMonitor-Setup-*.exe` 安装包即可。

`ProxyMonitor-portable.zip` 是便携版，适合不想安装、只想解压运行的用户。

## 注意

轻量测速用于节点之间的相对排名，不等于完整带宽测速。它的目标是尽量少影响日常上网。
