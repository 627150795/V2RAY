## 发现方式

- 固定数据目录：扫描各客户端在 `%APPDATA%`、`%LOCALAPPDATA%` 和用户配置目录中的常见位置。
- 便携版：从运行进程定位 Hiddify、NekoBox/NekoRay、sing-box 和 FlClash 的程序目录。
- 防误识别：忽略 v2rayN 内部启动的 sing-box 核心，不把代理核心当作独立客户端。

## 限制

- Mihomo 系仅在 REST API 或命名管道控制接口在线时执行延迟测试；接口离线仍可读取 YAML 节点和订阅来源。
- Mihomo 系切换需要准确选择策略组。当前版本不会猜测目标组，因此保持禁用。
- sing-box 系桌面客户端没有统一、稳定的控制 API。当前版本只读 JSON 配置，不启动核心、不测速、不切换。
- “结构测试”说明适配器通过离线配置夹具验证，不等同于在对应客户端每个发布版本上完成真实运行测试。
